Guía Didáctica

📘 Guía didáctica
This teaching guide helps teachers to implement the project "Farm Animals" in 2nd grade of Primary Education.

🎯 Relationship between activities and evaluation criteria.

ACTIVITY DESCRIPTION  EVALUATION CRITERIA
Session 1: Introduction

 Discovering the animals with a Power point presentation  Recognises basic vocabulary related to farm animals
Session 2: Vocabulary Learning and repeating animal names and its charcateristics. Playing games in pairs using wordwall. Recognises basic vocabulary and structures.
Session 3: Song Listen to the song and sing it Associates words with images and sounds
Session 4: Grammar (production) Completing simple sentences using genially Produces simple written/oral structures
Session 5: Guess the animal  Interactive games using canva Associates words with images
Session 6&7: Final Project! Prepare and show your lapbook  Participates in basic oral interaction using simple expressions


🧠 Key competences

- Linguistic communication
- Digital competence
- Learning to learn

📝 Recommendations for teachers

- Use visual support (flashcards, images)
- Repeat structures frequently
- Encourage participation in a safe environment
- Adapt activities to different learning rhythms

🌈 Attention to diversity

-Visual support
-Simple language
-Flexible tasks

🧠 DUA principles

- Multiple ways of learning (audio, images, games)
- Different ways to express learning (drawing, speaking, digital)
- Motivation through play and creativity
